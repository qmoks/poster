## Running Locally

```bash
git clone https://github.com/bozzhik/web-poster.git
cd web-poster
pnpm i
pnpm dev
```

### Full CLI command

```bash
git clone https://github.com/bozzhik/web-poster.git && cd web-poster && pnpm i && code .
```
